## Why use this template?

This template provides a carefully organized starting point with sensible defaults so teams can:

- Accelerate development with reusable modules and scaffolding.
- Keep code quality high with linting and CI-ready Gradle setup.
- Run local infra (Postgres, Keycloak) and sample apps with minimal setup.

Start with the Quick Start and the Developer Guide to get productive quickly with this template.
